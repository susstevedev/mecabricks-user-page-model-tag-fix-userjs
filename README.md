# mecabricks-user-page-model-tag-fix-userjs
Fix for [mecabricks]id[/mecabricks] on user comment pages.
# How to install
Install [TamperMonkey](https://www.tampermonkey.net) or [Violentmonkey](https://violentmonkey.github.io).
Then click [here](https://github.com/susstevedev/mecabricks-user-page-model-tag-fix-userjs/raw/main/index.user.js). You should see an installation prompt. Click install.
